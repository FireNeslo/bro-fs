# introduction
I think it would be neat to have a simple filesystem for the browser.

## Install

### Requirements
* [nodejs](nodejs.org)
* [bower](bower.io)
* [gulp](gulpjs.com)

``` bash
$ git clone https://github.com/FireNeslo/bro-fs.git
$ cd bro-fs
$ npm install
$ bower install
$ gulp
$ gulp watch &
$ python -m SimpleHTTPServer
```
